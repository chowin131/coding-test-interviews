# 외주 수익 최대화하기

**출처**: 삼성 SW 역량 테스트 2017 상반기 오전 2번
**난이도**: 평균 111분 / 정답률 40%

## 문제 요약

n일 휴가 동안 외주 작업으로 수익을 최대화한다.
- 각 날짜 i에 외주 작업 1개가 주어짐 (기한 t, 수익 p)
- i일에 시작해서 t일 동안 수행 → i~(i+t-1)일 사용
- 동시에 두 작업 수행 불가, 휴가 이후 작업 불가

## 풀이

**DP + 스케줄링**

`dp[j]` = j일까지 완료했을 때 최대 수익

i번째 작업 (1-indexed 시작일 `i`, 종료일 `i+t-1`):
1. 종료일이 n을 초과하면 수행 불가 → skip
2. 시작일 이전까지의 `dp` 최댓값을 구함
3. `dp[종료일]` 갱신

### 핵심 코드

```cpp
v[i] = {i + 1, i + len, p};  // {시작일, 종료일, 수익}

for (int j = 0; j < get<0>(v[i]); j++)
    temp = max(dp[j], temp);  // 시작일 이전 dp 최댓값

dp[get<1>(v[i])] = max(dp[get<1>(v[i])], temp + get<2>(v[i]));
```

## 복잡도

- 시간: O(n²)
- 공간: O(n)

## 제한 조건

- 1 ≤ n ≤ 15
- 1 ≤ t ≤ 5
- 1 ≤ p ≤ 1,000
