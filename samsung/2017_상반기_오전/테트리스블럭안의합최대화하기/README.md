## 문제 설명

n×m 격자판의 각 칸에 자연수가 적혀 있을 때, 5종류의 테트리스 블럭(자유 회전/반전 가능) 중 하나를 올려놓아 블럭이 덮는 칸의 합이 최대가 되는 값을 구하라.

### 제한 조건
- `4 ≤ n, m ≤ 200`
- `1 ≤ 자연수 ≤ 1,000`

## 풀이 방법

**완전탐색 - 블럭 하드코딩 + 대칭 변환**

5가지 블럭 각각에 대해 기준점 (i, j)를 격자 전체에 순회하면서 블럭을 놓아본다.

### 핵심 아이디어

블럭의 상대 좌표를 `Tr[type]`, `Tc[type]`에 미리 정의하고, `calr`, `calc` 부호 조합 4가지 `(+,+) (+,-) (-,+) (-,-)`로 4방향 회전을 표현:

```
calr[c] * Tr[type][body]  → 행 방향 부호 전환으로 상하 반전
calc[c] * Tc[type][body]  → 열 방향 부호 전환으로 좌우 반전
```

행·열을 **교환**한 버전도 따로 시도하여 90° 회전을 커버:
```
nr = i + calr[c] * Tc[type][body]   // Tr, Tc 교환
nc = j + calc[c] * Tr[type][body]
```

→ 부호 4가지 × (원본 + 전치) 2가지 = **8가지 변환**으로 모든 회전/반전 포함

### 시간복잡도

`O(5 × n × m × 8 × 4)` = `O(n × m)` ≈ 약 160만 연산
