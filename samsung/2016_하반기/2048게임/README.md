## 문제 설명

4×4 격자판에서 상하좌우 중 한 방향으로 블록을 밀면:
- 블록들이 해당 방향으로 이동
- 같은 숫자끼리 만나면 합산 (단, 한 번 합쳐진 블록은 같은 턴에 다시 합쳐지지 않음)
- 3개 이상 연속된 같은 숫자는 벽에 가까운 쪽부터 2개씩 합산

n×n 격자판이 주어질 때, **5번 이동** 후 얻을 수 있는 가장 큰 블록의 최댓값을 구하라.

### 제한 조건
- `1 ≤ n ≤ 20`
- 블록의 값: 2 이상 1024 이하 (2의 거듭제곱)

## 풀이 방법

**완전탐색 (백트래킹)**

5번의 이동에 대해 상하좌우 4방향을 모두 시도 → 최대 4^5 = 1024가지 경우의 수

각 이동 구현 시 핵심:
1. 이동 방향의 벽에서부터 순서대로 처리
2. `flag` 배열로 이미 합쳐진 블록 표시 (중복 합산 방지)
3. 빈 칸(0)을 건너뛰며 이동 위치 결정

### 시간복잡도

`O(4^5 × n^2)` = `O(1024 × 400)` ≈ 약 40만 연산

## 예제

| 입력 | 출력 |
|------|------|
| n=4, `[[4,2,0,8],[4,2,2,4],[0,8,8,2],[4,2,2,2]]` | `32` |
| n=2, `[[8,16],[16,8]]` | `16` |
| n=4, `[[8,16,0,0],[0,0,16,8],[0,0,0,0],[0,0,0,0]]` | `32` |
