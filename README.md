# 코딩 테스트 & 인터뷰 준비 📚

코딩 테스트 문제 풀이 기록 저장소입니다.  
문제 정보, 풀이 방법, 난이도, 회고 등을 체계적으로 관리합니다.

---

## 📁 디렉토리 구조

```
coding-test-interviews/
├── programmers/       # 프로그래머스 문제 풀이
├── baekjoon/          # 백준 문제 풀이
├── leetcode/          # LeetCode 문제 풀이
└── concepts/          # 알고리즘 개념 정리
```

---

## 📝 문제 풀이 기록 방법

각 플랫폼 디렉토리 아래에 **난이도 → 문제 폴더** 형태로 정리합니다.

예시:
```
programmers/
└── level2/
    └── 타겟넘버/
        ├── solution.py
        └── README.md
```

각 문제 폴더의 `README.md`는 [TEMPLATE.md](./TEMPLATE.md)를 참고해 작성합니다.
